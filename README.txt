Si vous lisez ceci c'est que vous avez besoin d'aide. Pas de panique ce petit guide est fait pour vous.

Outil Crayon
Changer la taille du pinceau : Style -> Pen -> Width 
Changer le tracé : Style -> Pen -> Line
Changer la couleur : Style -> Pen -> Color

Outil Remplissage
Changer le remplissage : Style -> Brush -> Fill
Changer la couleur : Style -> Brush -> Color

Raccourci clavier : 
Ctrl -s : Sauvegarder un dessin
Ctrl Shit-s : Sauvegarder un dessin comme
Ctrl -n : Nouvelle page blanche
Ctrl -o : Ouvrir un dessin
Ctrl -F : Menu file
Ctrl -F : Menu tool
Ctrl -F : Menu style
Ctrl -F : Menu help

Sélectionner plusieurs items graphiques : Clic sur un endroit du dessin -> Faites passer votre curseur sur les items en maintenant le clic droit -> Relacher le clic 
Deplacer un objet : Clic droit sur un objet -> Deplacer l'objet en maintenant le clic droit -> Relacher le clic à l'endroit voulu
Creer un polygone : 
Tools -> Polygon
Clic droit pour créer un point -> Double clic pour fermer la figure 
Enjoy your draw !
